## Problem Statement
A university department is tracking student performance across courses manually using paper records.
Teachers have to dig through these records to find a student, update grades, or get a sense of how the class is performing overall.
It is error-prone and slow, and there is no single place to look up the current state of the class.

## Requirements

[] Let teachers to add a new student to the class roster by name.
[] Let teachers to remove a student who has left the course.
[] Display the full class roster on demand.
[] Let teachers to record a GPA for a student between 0.0 and 10.0.
[] Display the class average GPA alongside the roster.
